# 🛒 HaatBazar - Multi-vendor E-commerce Marketplace

A full-stack multi-vendor marketplace built with Laravel & Blade.

## 🚀 Tech Stack
- **Backend:** Laravel 12
- **Frontend:** Blade Template
- **Database:** MySQL
- **Payment:** bKash, Nagad, Card, COD

## ✨ Features
- Multi-vendor shop system
- Product & category management
- Order & payment management
- Admin, Seller & Buyer roles
- Coupon & discount system
- Product reviews & wishlist

## ⚙️ Installation
```bash
git clone git@github.com:HASIBULALAMH/haatbazar.git
cd haatbazar
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
